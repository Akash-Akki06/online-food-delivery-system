🍔 Online Food Delivery System
📌 Project Description

This is a DBMS mini project developed using Flask and MySQL.
The application allows users to register, login, browse restaurants, view menus, add items to cart, and place orders.

🎯 Objectives
To understand database integration with web applications
To implement user authentication system
To manage restaurant and food data using MySQL
To simulate a real-world food ordering system

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: MySQL
Tools: VS Code, GitHub

⚙️ Features
🔐 User Registration & Login
🍽️ Restaurant listing based on city
📋 Menu display for each restaurant
🛒 Add to Cart functionality
💰 Total price calculation
📦 Order placement with confirmation

🗄️ Database Tables
users → stores user details
restaurants → stores restaurant information
food_items → stores menu items

📚 Conclusion

This project demonstrates how a food delivery system works using database management concepts and web development. It helped in understanding real-world application design and database interaction.
